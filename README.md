# 가계부 33차 — 앱 스타일 UI

## 디자인

- 생성한 UI 콘셉트의 흰색 카드, 연한 배경, 보라색 포인트, 둥근 목록 스타일 반영
- 헤더·동기화 상태·월 선택기·페이지 제목을 모바일 앱처럼 간결하게 정리
- 모든 메뉴의 카드, 목록, 필터, 버튼, 달력, 통계, 설정 화면을 같은 디자인 언어로 통일
- 모바일 하단 메뉴를 화면 가장자리와 떨어진 둥근 플로팅 메뉴로 변경
- 입력 모달과 각종 동작 버튼도 동일한 앱 스타일로 정리
- 다크모드 기능 유지

## 상단 월 요약

- 월 남은 예산, 월 소득, 월 소비, 월 저축, 월 카드 결제예정은 All 탭에서만 표시
- 각 항목에 색상별 원형 아이콘과 상세 설명, 이동 화살표 스타일 적용
- 월 예산 기준 수정 버튼을 요약 제목줄에 배치
- 다른 탭으로 이동하면 월 요약 전체가 숨겨져 해당 메뉴 내용에 집중 가능

## 기능 보존

- 32차의 계산, 입력, Firebase 동기화, 검색, 통계, 백업, 접기 상태 저장 기능 유지
- 기존 데이터 구조와 백업 파일 호환
- 브라우저 탭, 실행화면, 설정, 백업 내부 버전, PWA 캐시를 모두 33차로 통일
