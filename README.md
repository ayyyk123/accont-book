# 가계부 35.2차 — 통장 이체 복원

## All 탭

- 월 총내역 상단에 `+ 통장 이체` 버튼 복원
- 버튼 위치: `+ 통장 지출` 바로 오른쪽
- 기존 통장 이체 입력창을 그대로 사용하므로 보내는 통장, 받는 통장, 금액과 날짜 입력 가능

## 글꼴 시험 제거

- 백업·설정의 감따러가요 글꼴 시험 항목 제거
- Bakey 로컬 글꼴을 확인하던 JavaScript와 전용 CSS 제거
- 관련 로컬 저장값 처리와 상태 안내 문구 제거
- 기존 Apple 시스템 글꼴 사용

## 유지 기능

- 35.1차 목적자금 결과표 대분류·소분류 필터 유지
- 상단 5개 월 요약은 All 탭에서만 표시
- 35차 목적자금 기간조회와 34차 카드값 기간조회 유지
- Firebase 데이터, 백업 구조와 접기 상태 저장 방식 유지
- 화면·백업·PWA 캐시를 모두 35.2차로 통일
