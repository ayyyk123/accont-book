# 가계부 32차 — 코드 정리·안정화 버전

## 화면 수정

- 모바일 월 선택기 오른쪽의 파란 이번 달 버튼을 세로 정중앙으로 정렬
- 모든 접기 가능 메뉴는 저장 이력이 없을 때 기본적으로 접힌 상태로 시작
- 사용자가 마지막으로 펼치거나 접은 상태는 다음 실행에도 그대로 유지
- 예전 버전의 접기 상태 저장값도 읽어서 기존 사용 상태를 이어받음
- 백업·설정의 제목줄 기능 버튼과 접기 버튼을 분리하여 그대로 유지
- 월 소득·소비 달력과 고정항목 달력의 모바일 7열 화면 이탈 방지 유지

## 내부 코드 정리

- 누적된 임시 CSS 4개를 제거하고 32차 공통 스타일 하나로 통합
- 누적된 임시 JavaScript 3개를 제거하고 공통 접기 시스템 하나로 통합
- 화면 실행 후 DOM을 반복 포장하던 접기 코드를 제거하고 HTML 구조 자체에 접기 본문을 구성
- 일반 메뉴와 백업·설정 메뉴의 서로 다른 접기 방식을 하나로 통일
- 달력 모드 선택 버튼의 data 속성 판정 오류를 최종 함수에서 직접 수정
- 기존 접기 상태 저장소와 새 저장소를 자동으로 호환
- 총 24개 화면 메뉴와 15개 설정 항목을 공통 구조로 변환

## 버전

브라우저 탭, 실행화면, 설정, 백업 파일명·내부 버전, PWA 캐시를 모두 32차로 통일했습니다.
