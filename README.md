# 가계부 36.2차 — 통장목록 이체 버튼 수정

## 오류 원인

통장목록 이체 버튼에 `button.onclick` 프로퍼티로 클릭 함수를 연결했지만,
통장 카드 화면을 HTML 문자열로 다시 만드는 과정에서 함수가 사라졌습니다.
그래서 버튼은 보이지만 눌러도 반응하지 않았습니다.

## 수정 내용

- 통장목록 이체 버튼을 문서 이벤트 위임 방식으로 변경
- 이체 버튼을 눌렀을 때 부모 통장카드의 잔액 화면이 열리지 않도록 차단
- 새로 생성되는 버튼 HTML에도 실제 `onclick` 속성을 추가해 이중 보호
- 해당 통장을 보내는 통장으로 자동 선택
- 받는 통장이 같으면 다른 통장으로 자동 선택
- 입력창 제목에 선택한 통장명 표시

## 확인한 동작

Chromium에서 실제 버튼을 눌러 다음을 확인했습니다.

- 이체 버튼 클릭 인식
- 부모 통장카드 클릭 차단
- 통장 이체 입력창 열림
- 거래유형 `transfer`
- 보내는 통장 자동 선택
- 받는 통장 자동 선택
