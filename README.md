# 가계부 29.4차

29.3차의 통장 고정지출 취소, 탭 유지, 카드값 월 연동 기능을 유지하면서 고정항목의 이번 달 실제 금액 수정 기능을 추가했습니다.

## 29.4차 변경사항

- 고정수입 `입금 확인` 시 이번 달 실제 입금액 수정 가능
- 통장 고정지출 `납부 확인` 시 이번 달 실제 출금액 수정 가능
- 카드 고정지출 `승인 확인` 시 이번 달 실제 승인금액 수정 가능
- 수정 금액은 해당 월의 소득·소비·남은 예산·카드 결제예정·통장 잔액에 함께 반영
- 다음 달부터는 등록된 원래 고정금액으로 자동 생성
- 이미 확인 완료한 항목도 `금액 수정` 버튼으로 다시 조정 가능
- 앱 제목·설정·백업·PWA 캐시 버전을 29.4차로 통일

## 사용 전 권장사항

업데이트 전 기존 앱에서 전체 백업 파일을 한 번 저장한 뒤 새 파일을 배포하세요.
