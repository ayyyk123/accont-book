# 가계부 29.8차

## 모바일 입력창 개선

- 카드 사용 입력을 포함한 모든 입력 모달에 적용
- 긴 입력창은 모달 내부에서 부드럽게 스크롤
- 저장·취소 버튼을 모달 아래쪽에 고정
- 아이폰 하단 안전영역만큼 버튼 여백 확보
- 모바일 키보드가 올라오면 실제 보이는 화면 높이에 맞춰 모달 크기 자동 조정
- 입력 중인 항목이 키보드에 가려지면 화면 가운데로 자동 이동
- 모바일 입력글자 크기를 16px로 설정해 아이폰 자동 확대 방지
- 가로·세로 회전 시 모달 높이 다시 계산
- 브라우저 탭, 실행화면, 가계부 이름 옆, 설정, 백업명, PWA 버전을 모두 29.8차로 통일
