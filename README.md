# 가계부 36.7차 — 카드 대신 결제·받을 돈 연동

## 카드 사용 입력

카드 사용 입력창에 `나중에 돈으로 받을 금액 있음`을 추가했습니다.

입력 항목:

- 받을 사람·기관
- 받을 금액
- 받기로 한 날
- 카드 총 구매금액과 받을 돈 처리 방식 미리보기

월 총내역과 빠른 입력에는 `+ 대신 결제` 버튼을 추가했습니다.

## 기록 방식

예: 내 카드로 100,000원을 결제하고 친구에게 40,000원을 받는 경우

- 카드 사용금액: 100,000원
- 카드 결제예정액: 100,000원
- 받을 돈 자동등록: 40,000원
- 받을 돈 상대방과 약속일 자동 연결

카드값은 실제 결제액 전체를 유지하고, 받을 금액은 받을 돈으로 따로 관리합니다.

## 입금받기

받을 돈 화면에서 전액 또는 일부 금액을 입금 처리할 수 있습니다.

- 선택한 통장 잔액 증가
- 통장 입금으로 기록
- 월 소득에서는 제외
- 일부 수령 후 남은 받을 돈 자동 계산
- All 탭과 통장 전체 거래내역에서 확인 가능

## 자동 동기화

- 카드 사용을 수정하면 연결된 받을 돈도 수정
- 받을 돈에서 상대방·금액·약속일을 수정하면 카드 사용과 연결 유지
- 아직 입금 처리하지 않은 상태에서 체크 해제 시 자동 받을 돈 삭제
- 연결된 받을 돈 삭제 시 카드 사용은 유지
- 카드 사용 삭제 시 미처리 받을 돈은 함께 삭제
- 이미 입금받은 기록이 있으면 카드 사용을 삭제해도 받을 돈·입금 기록은 독립 기록으로 보존

## 표시

- 카드 사용내역에 받을 사람·총 받을 금액·남은 금액 표시
- 모두 받으면 `돌려받기 완료` 표시
- 받을 돈 목록에 `카드 대신 결제` 배지 표시
- 검색에서 상대방과 받을 돈 상태 검색 가능

## 데이터 호환

- 기존 카드 사용·받을 돈 데이터 유지
- Firebase 동기화와 로컬 백업에 새 연결정보 포함
- 카드 대신 결제 입금은 재접속 후에도 통장 입금·소득 제외 상태 유지

## 유지 기능

- 36.6.1차 투자 평가기준일 보존 수정
- 미완료·오늘 완료 이체 알림과 접기·펴기
- 대출별 내역과 통장 전체 거래장부
- 모든 통장의 이체 예정·거래내역 버튼
- 한글 입력 수정
- 월·일 선택 조회
- All 고정필터
- 투자 독립 탭
