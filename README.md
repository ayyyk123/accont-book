# 가계부 27.7차

## 이번 수정

- 월 소비달력에서 소비가 있는 날짜를 누르면 그날의 카드 사용과 통장 직접 지출이 한 목록으로 표시됩니다.
- 날짜 상세의 각 항목을 누르면 카드 사용 수정 또는 통장 지출 수정 화면으로 바로 이어집니다.
- 홈의 월 소비내역과 월 소비내역 탭의 행을 눌러도 해당 원본 수정화면으로 이동합니다.
- 카드값의 카드별 상세에서 개별 사용처를 누르면 카드 사용 수정화면으로 이어집니다.
- 상단 월 소득·소비·저축·예산 상세의 개별 항목도 가능한 경우 원본 수정화면으로 연결했습니다.
- 홈의 통장 상세에서는 소득·직접지출·이체를 누르면 통장내역 수정화면으로, 카드값 출금은 카드값 화면으로 연결됩니다.
- 받을 돈·줄 돈 상세 항목을 누르면 해당 금전메모 수정화면으로 이어집니다.
- 다음 달 고정항목 예정 행을 누르면 해당 고정항목 수정화면으로 이어집니다.

## 이전 기능 유지

- 카드와 통장 소비 월별 통합
- 카드별 결제예정 상세
- 네모 박스 접기·펴기 및 상태 저장
- 이전 달·다음 달·이번 달 빠른 이동
- 내 예산과 목적자금 남은 금액 분리
- 일반대출·마이너스대출 이자 고정지출 자동 반영
