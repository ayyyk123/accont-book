# 가계부 29차

28.2차를 기준으로 목적자금 충전 방식을 실제 수령 내역 중심으로 변경했습니다.

## 주요 변경
- 기존 `충전 받음 / 아직 못 받음` 버튼 제거
- 목적자금별 실제 받은 날짜·금액을 여러 건으로 나누어 등록
- 실제 수령 내역 수정·삭제 지원
- 입금 통장을 선택하면 통장잔액에 입금 반영
- 목적자금 수령액은 월 소득과 생활비 예산에서 제외
- 예정 충전액, 실제 수령액, 미수령액, 월 사용액, 실제 가용잔액 표시
- 실제 수령액보다 목적자금 사용액이 많으면 부족분을 받을 돈으로 자동 기록
- 이후 실제 수령액을 추가하면 받을 돈이 자동 감소
- 수령내역·마이그레이션 기준·부분수령 정보 백업 포함
- 28.2차의 모든 출금 보충 이체, 마이너스통장 우선순위 기능 유지

## 기존 데이터
29차로 처음 실행한 달의 시작 잔액은 기존 목적자금 계산값을 이월 기준으로 보존합니다. 이후부터는 실제로 입력한 수령액을 기준으로 계산합니다.
