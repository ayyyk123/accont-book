# 가계부 29.7차

## 변경사항

- 고정저축 `이체 완료` 시 이번 달 실제 저축금액 입력 가능
- 기본 고정저축 금액과 다르게 입력해도 선택한 달에만 적용
- 완료 후에도 `금액 수정` 버튼으로 다시 변경 가능
- 수정금액이 이번 달 남은 예산과 고정저축 예정·완료 금액에 반영
- 출금통장과 입금통장 잔액에도 수정금액으로 반영
- 목적자금과 연결된 고정저축도 실제 수정금액으로 반영
- 다음 달부터는 원래 등록한 고정저축 금액으로 자동 복귀
- 기존 `이번 달 취소`와 `취소 해제` 기능 유지
- 브라우저 탭, 실행화면, 가계부 이름 옆, 설정, 백업명, PWA 버전을 모두 29.7차로 통일
