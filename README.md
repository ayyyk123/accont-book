# 가계부 27.4차

## 이번 수정

- 각 탭의 네모 박스마다 `접기`·`펴기` 버튼을 추가했습니다.
- 고정수입·지출·저축 탭처럼 한 박스 안에 목록 박스가 여러 개 있는 화면은 안쪽 박스도 각각 접을 수 있습니다.
- 설정 탭의 개별 안내 박스도 각각 접고 펼칠 수 있습니다.
- 접어둔 상태는 현재 브라우저에 저장되어 새로고침하거나 다시 열어도 유지됩니다.
- 상단 월 선택기의 왼쪽에 `이전 달`, 오른쪽에 `다음 달` 버튼을 배치했습니다.
- `이번 달` 버튼도 유지했습니다.

## 이전 기능 유지

- 내 예산 남음과 목적자금 남음을 분리해 표시합니다.
- 카드 결제예정액을 카드별로 확인하고 카드별 세부내역을 열 수 있습니다.
- 일반대출·마이너스대출 이자를 지정한 시작월부터 고정지출로 반영합니다.

GitHub 저장소 최상위 파일을 이 압축파일의 파일들로 덮어쓴 뒤 배포하면 됩니다.
