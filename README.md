# 가계부 36차 — All 고정필터·통장목록 이체

## All 탭 기간

- 시작월과 종료월을 항상 같은 위치에 표시
- 시작월은 상단 선택월로 최초 설정
- 종료월은 기본 미선택
- 종료월이 비어 있으면 시작월 한 달만 조회
- 종료월을 선택한 경우에만 시작월부터 종료월까지 기간조회
- 종료월을 지워 한 달 조회로 바로 복귀 가능

## 고정 필터 구조

선택에 따라 필터가 생기거나 사라지지 않고 항상 같은 자리에 표시됩니다.

- 거래유형
- 결제구분
- 카드
- 통장
- 대분류
- 소분류
- 목적자금
- 수령상태
- 검색

### 거래유형 중복선택

아래 거래유형을 여러 개 동시에 선택할 수 있습니다.

- 소득
- 지출
- 통장입금
- 통장출금

### 선택 불가능한 항목

각 필터 버튼은 항상 유지됩니다.
필터를 열면 현재 기간·거래유형·다른 조건에서 사용할 수 없는 항목은
사라지지 않고 회색으로 비활성화됩니다.

예:
- 소득·입금·출금만 선택하면 카드 선택 비활성화
- 입금·출금만 선택하면 대분류·소분류 비활성화
- 목적자금을 선택하지 않으면 수령상태 비활성화
- 카드 또는 통장을 고르면 결제구분이 자동으로 맞춰짐

## 목적자금

- 목적자금을 별도 화면 모드로 사용하지 않음
- 일반 All 결과 안에서 목적자금 필터로 조회
- `목적자금 사용만` 또는 개별 목적자금 선택 가능
- 목적자금을 선택하면 수령상태 필터 사용 가능
- 목적자금 선택 시에도 필터 배열과 조회 화면은 바뀌지 않음

## 통장,카드관리

각 통장 카드의 빠른 입력 버튼:

- 입금
- 출금
- 지출
- 소득
- 이체

`이체`를 누르면 해당 통장이 보내는 통장으로 자동 선택됩니다.

## 유지 기능

- 표 제목의 날짜·결제수단·항목 등 세부 열 필터
- 투자 독립 탭과 평가기준일
- 통장 입출금·현금계좌·투자거래 연결
- 목적자금 수령상태 계산
- 카드값 기간조회
- Firebase 데이터와 백업 구조
