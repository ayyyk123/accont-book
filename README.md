# 가계부 29.2차

29차 목적자금 계산 기준을 장부잔액과 실제 수령 현황으로 분리했습니다.

## 핵심 수정
- 목적자금 장부잔액: 전월 이월 + 이번 달 충전예정액 - 이번 달 사용액
- 실제 수령 현황: 예정액, 실제 수령액, 미수령액을 별도 표시
- 월중 받을 돈: 이번 달 충전예정액 - 실제 수령액
- 월말 확정 받을 돈: 이번 달 사용액 - 실제 수령액(최소 0원)
- 미래월은 받을 돈을 미리 생성하지 않음
- 시작금액을 0원으로 수정했는데도 이전 마이그레이션 잔액이 남던 오류 제거
- 실제 받은 날짜·금액 분할입금, 수정·삭제, 통장 반영 기능 유지
- 28.2차와 29차의 카드값 보충·마이너스통장·백업 기능 유지


## 29.2차 변경사항
- 앱 상단·실행화면·설정·백업·PWA 버전 표시를 29.2차로 통일
- 카드 고정지출에 `이번 달 취소` 버튼 추가
- 이번 달 취소 시 해당 월 소비·예산 차감·카드 결제예정에서만 제외
- 고정항목 자체와 다음 달 이후 자동반영은 유지
- 취소된 항목은 `이번 달 취소` 상태로 표시하며 `취소 해제` 가능
- 카드 고정지출의 `납부 확인` 명칭을 `승인 확인`으로 명확화
