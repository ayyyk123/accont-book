# 가계부 36.1차 — 월 선택 후 일 선택

## All 조회기간

- `시작월` 문구를 `조회월`로 변경
- 종료월의 `선택 시 기간 적용` 안내를 제목 옆으로 이동
- 조회월·종료월 입력칸 높이를 다른 필터와 맞춤
- 월을 먼저 선택한 뒤 같은 칸 오른쪽의 `일 선택`을 눌러 날짜 지정
- 일을 고르지 않으면 기존처럼 월 전체 조회

### 조회 규칙

- 조회월만 선택하고 일 미선택: 해당 월 전체
- 조회월의 일까지 선택하고 종료월 미선택: 해당 하루
- 종료월 선택, 일 미선택: 종료월 말일까지
- 조회월·종료월의 일을 모두 선택: 정확한 시작일~종료일 기간
- 시작일과 종료일이 반대면 자동으로 순서를 맞춤

## 일 선택이 추가된 조회칸

- 상단 조회월
- All 조회월·종료월
- 카드값 시작월·종료월
- 통장내역 조회월
- 고정항목 월별 달력 조회월
- 통계 월별 조회월

고정항목 적용 시작월, 대출이자 시작월처럼 실제 설정값이 월 단위인 입력칸은
기존 월 단위를 유지합니다.

## 화면별 적용

- 상단 조회월: 선택한 날짜까지 소득·소비·저축·카드값·예산 누적 표시
- All: 선택한 정확한 날짜 범위로 거래 필터링
- 카드값: 카드별 결제일을 기준으로 날짜 범위 필터링
- 통장내역: 일을 선택하면 해당 날짜 거래만 표시
- 고정항목 달력: 선택한 날짜를 바로 선택하여 상세 표시
- 통계: 월별 모드에서 일을 선택하면 해당 하루만 집계

## 유지 기능

- All 고정 필터와 중복 거래유형 선택
- 표 제목의 세부 열 필터
- 투자 독립 탭과 평가기준일
- 통장목록 입금·출금·지출·소득·이체
- 통장 입출금·투자거래 연결
- Firebase 데이터와 백업 구조
