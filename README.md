# 가계부 31.3차

## 선택 버튼과 통계
- All 달력의 소득+소비, 소비만, 소득만 버튼 선택 색상을 확실하게 표시
- 고정항목 달력의 전체 고정 달력, 월별 보기 버튼 선택 색상을 확실하게 표시
- 통계 막대그래프의 각 대분류를 누르면 해당 대분류 상세 통계 표시
- 원형그래프 조각과 범례를 눌러도 해당 대분류 상세 통계 표시
- 소비·소득 전체 상세 통계 버튼 유지

## 고정항목 달력
- 월별 보기를 선택하면 독립적인 월 선택칸 표시
- 상단 월을 변경해도 월별 달력의 조회 월이 함께 변경
- 전체 고정 달력은 계속 기본 화면으로 유지

## PC 배열
- 현재 앱 버전과 Firebase 자동 동기화를 각각 한 줄에 하나씩 표시
- 백업·설정의 모든 안내 항목을 한 줄에 하나씩 표시
- AI 자산비서 분석 카드도 한 줄에 하나씩 표시

## 모바일 배열
- 접기·펴기 버튼과 설명 버튼의 크기와 높이를 축소
- 노란 편지 아이콘을 작고 연한 노란 확성기 아이콘으로 변경
- 수정·삭제·ON/OFF 등 동작 버튼 높이를 축소
- 고정항목 목록의 버튼이 겹치지 않도록 메타정보와 동작 버튼을 재배치
- 하단 메뉴가 화면 내용의 버튼에 가려지지 않도록 여백과 레이어 순서 보정

## 분류 항목 관리
- 지출 대분류와 소득 대분류를 각각 가로 스크롤 탭으로 표시
- 식비, 생활, 자기계발, 교통 등 대분류를 누르면 해당 대분류의 소분류만 아래에 표시
- 대분류 수정, 소분류 추가, 삭제 기능 유지

## 기타
- `저축목록`을 `저축 목록`으로 수정
- 브라우저 탭, 실행화면, 설정, 백업 파일명과 내부 버전, PWA 캐시를 모두 31.3차로 통일
