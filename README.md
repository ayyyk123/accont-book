# 가계부 27.8차

## 이번 수정

- 홈 상단의 월 소비를 누르면 카드·통장별 상세거래가 바로 펼쳐지지 않고, 먼저 각 카드와 각 통장의 월 소비 합계가 결제수단별로 표시됩니다.
- 결제수단을 누르면 해당 카드 또는 통장의 실제 소비내역이 표시되고, 각 항목을 누르면 원본 수정화면으로 이어집니다.
- 결제수단별 목록에서 내 예산 사용액과 목적자금 사용액을 나누어 확인할 수 있습니다.
- 월 소비내역 탭의 결제수단 오른쪽에 `예산 구분` 열을 추가했습니다. 카드 사용은 `내 예산` 또는 `목적자금 · 이름`으로 표시되고, 통장 직접 지출은 현재 구조에 따라 `내 예산`으로 표시됩니다.
- 월 소비달력의 날짜 상세에도 내 예산·목적자금 구분이 함께 표시됩니다.

## 이전 기능 유지

- 월 소비달력 날짜 클릭 및 수정화면 연결
- 카드와 통장 소비 월별 통합
- 카드별 결제예정 상세
- 네모 박스 접기·펴기 및 상태 저장
- 이전 달·다음 달·이번 달 빠른 이동
- 내 예산과 목적자금 남은 금액 분리
- 일반대출·마이너스대출 이자 고정지출 자동 반영
