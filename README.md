# 가계부 8차 — Firebase 자동 동기화 · 차입금 자동등록

## 추가된 기능
- Firebase 이메일 회원가입·로그인·로그아웃
- 로그인 계정별 Firestore 자동 저장
- PC와 휴대폰 사이 실시간 동기화
- 기존 로컬 자료를 처음 만든 계정에 자동 업로드
- 통장 지출에서 `대여금` 선택 시 받을 돈 자동등록
- 통장 입금에서 `차입금` 선택 시 줄 돈 자동등록
- 받을 돈·줄 돈 모두 0원 완료 가능
- 원래 금액, 실제 처리금액, 면제금액 기록 유지

## 사용 전 확인
1. 인터넷 연결 상태에서 `index.html`을 엽니다.
2. 회원가입 또는 로그인합니다.
3. 상단에 `자동 동기화 중`이 표시되는지 확인합니다.
4. 다른 기기에서도 같은 이메일로 로그인합니다.

## 주의
- 처음 로그인할 때 클라우드에 기존 자료가 있으면 클라우드 자료를 우선 불러옵니다.
- 중요한 변경 전에는 백업 JSON을 내려받아 보관하는 것이 안전합니다.
- 로컬 HTML 파일은 모바일에서 실행이 번거로울 수 있으므로, 이후 Firebase Hosting에 배포하면 주소 하나로 편하게 접속할 수 있습니다.
