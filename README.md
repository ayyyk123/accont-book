# 가계부 31.6차

## 모바일 달력 폭 수정

- 월 소득·소비 달력의 7개 요일 열을 모바일 화면 안에서 균등 분할
- 각 날짜 칸에 `min-width: 0`과 최대 폭 제한을 적용해 일요일 칸이 화면 밖으로 밀리지 않도록 수정
- 달력 전체와 달력 섹션의 가로 넘침을 차단
- 날짜별 소득·소비 금액과 건수는 칸 안에서 자동 줄바꿈
- 작은 아이폰 화면에서는 날짜 칸 간격, 안쪽 여백, 글자 크기를 추가 축소
- 고정항목 달력에도 같은 7열 폭 제한 적용
- 기존 31.5차의 백업·설정 제목줄 버튼 배치와 모든 기능 유지
- 브라우저 탭, 실행화면, 설정, 백업 파일명·내부 버전, PWA 캐시를 모두 31.6차로 통일
