# 가계부 27.5차

## 이번 수정

- 홈의 `최근 소비`가 전체 기간의 최신 카드 사용을 보여주던 문제를 수정했습니다.
- 홈에서는 상단에서 선택한 월의 카드 사용과 통장 직접 지출만 함께 표시합니다.
- `카드 사용` 탭을 `월 소비내역`으로 변경했습니다.
- 월 소비내역에서 카드 사용과 통장 직접 지출을 날짜순으로 한 번에 확인할 수 있습니다.
- `카드+통장 전체`, `카드만`, `통장 지출만` 필터를 추가했습니다.
- 카드·통장별 건수와 금액 합계를 함께 표시합니다.
- 월 소비 달력도 카드와 통장 지출을 합산해 표시합니다.
- 홈의 `전체 보기`를 누르면 홈에서 선택한 월이 그대로 월 소비내역에 적용됩니다.

## 이전 기능 유지

- 각 네모 박스 접기·펴기와 상태 저장
- 이전 달·다음 달·이번 달 버튼
- 카드별 결제예정 상세
- 내 예산 남음과 목적자금 남음 분리 표시
- 일반대출·마이너스대출 이자 고정지출 자동 반영
