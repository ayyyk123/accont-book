# 가계부 35차 — 목적자금 조회·Bakey 글꼴 시험

## All 탭 목적자금 조회

- 결제수단 필터에 `목적자금만` 추가
- 시작월과 종료월을 선택해 여러 달의 목적자금 사용내역 조회
- 목적자금별 선택 가능
- `수령상태 전체 / 미수령 있음 / 수령완료` 필터 제공
- 카드 사용과 통장 직접지출 중 목적자금으로 지정한 거래를 함께 표시
- 총 사용금액, 실제 받은 금액, 아직 받을 금액과 사용 건수 자동 계산
- 월별 사용·수령·미수령 합계 표시
- 검색어, 대분류, 소분류 필터와 함께 사용 가능
- 수령상태는 같은 목적자금·같은 달의 실제 수령액을 사용일 순서로 배정해 계산

## And감따러가요Plus 글꼴 시험

- Bakey에서 아이폰에 활성화된 로컬 글꼴을 `local()` 방식으로 불러오도록 설정
- 다음 이름을 순서대로 시험:
  - And감따러가요Plus M L Light
  - And감따러가요Plus
  - And감따라가요Plus M L Light
  - And감따라가요Plus
- 백업·설정의 `감따러가요 글꼴 시험`에서 사용 여부와 인식 상태 확인
- Safari/PWA가 해당 로컬 글꼴을 제공하지 않으면 기존 시스템 글꼴로 자동 표시
- 글꼴 시험을 설정에서 끄고 켤 수 있으며 선택은 현재 기기에 저장
