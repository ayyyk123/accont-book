# 가계부 31.2차

## All 탭
- 통장만 선택 시 특정 통장을 추가로 선택해 조회 가능
- 소득+소비, 소비만, 소득만 달력 버튼의 선택 상태를 색상으로 명확히 표시
- 모바일 월 선택칸이 옆 필터를 침범하지 않도록 폭 조정
- 모바일 내역을 날짜·구분·항목·금액·관리버튼이 짧게 배치되는 간결한 줄목록으로 재구성

## 접기와 설명
- 모든 접기·펴기 버튼을 `+`, `−`로 변경
- 모든 접기·펴기 버튼 위치를 오른쪽 상단으로 고정
- 안내 설명과 주의 문구는 노란 편지 아이콘 안에 기본 접힘
- 노란 아이콘을 누르면 설명이 열리고 다시 누르면 접힘

## 고정내역
- 고정항목 달력의 기본 화면을 전체 고정항목 달력으로 변경
- 매월 등록일 1~31일 기준으로 모든 고정수입·고정지출·고정저축 표시
- `월별 보기` 버튼으로 선택한 달의 완료·대기·취소 상태 확인 가능
- 고정항목 달력은 기본 펼침 상태

## 카드값
- 카드별 결제예정액에 `+`, `−` 접기·펴기 버튼 추가
- 버튼은 카드별 오른쪽 상단에 고정
- 카드별 상세 결제내역과 납부 버튼을 접어서 확인 가능

## 목록 화면
- 대출,저축 탭의 `고정저축 목록`을 `저축목록`으로 변경
- 금전거래 탭을 간결한 줄목록으로 변경
- 통장,카드관리 탭의 카드목록을 간결한 줄목록으로 변경
- 검색,자산비서와 백업,설정 탭도 연결된 줄목록 형태와 간격으로 정리

## 분류,통계
- 막대그래프의 각 대분류를 누르면 해당 대분류의 상세 통계 표시
- 원형그래프 조각과 범례에서도 동일하게 대분류 상세 통계 표시
- 소비 대분류와 소득 대분류에 각각 `전체 상세 통계` 버튼 추가
- 모바일 상세 통계도 세로 길이가 짧은 줄목록으로 변경

브라우저 탭, 실행화면, 설정, 백업 파일명과 내부 버전, PWA 캐시를 모두 31.2차로 통일했습니다.
