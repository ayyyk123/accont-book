# 가계부 34차 — 카드값 기간 조회

## 카드값 탭

- 시작월과 종료월을 선택해 기간별 카드 결제예정액 조회
- 선택 기간의 전체 카드값 합계, 미확인 금액, 조회 카드 수와 개월 수 표시
- 검색 결과는 카드별 총액 목록으로 표시
- 카드 이름 또는 `+` 버튼을 누르면 선택 기간 안의 월별 카드값 표시
- 각 월을 누르면 사용기간, 결제 상세내역, 할부 회차와 납부 상태 확인
- 월별로 납부 완료 또는 예정으로 되돌리기 가능
- `이번 달` 버튼으로 현재 선택 월을 시작월·종료월에 즉시 적용
- 기존 단일 월 납부 처리, 카드 결제주기와 통장 잔액 반영 방식 유지

## 호환성

- 32차의 UI, 기본 접기, 마지막 접기 상태 저장, Firebase 동기화와 백업 구조 유지
- 기존 가계부 데이터와 백업 파일 호환
- 브라우저 탭, 설정, 백업 내부 버전과 PWA 캐시를 모두 34차로 통일
