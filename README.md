# 가계부 28차

## 이번 수정

- 홈의 `통장별 잔액`을 누르면 현재 잔액, 선택한 월의 잔액 미반영 고정항목, 모두 반영한 뒤의 예상잔액을 함께 보여줍니다.
- 통장별로 미반영 예정항목이 몇 건인지, 들어올 금액과 나갈 금액이 각각 얼마인지 표시합니다.
- `잔액 미반영 예정항목`을 누르면 고정수입·고정지출·고정저축 목록이 나오고, 각 항목을 누르면 해당 고정항목 수정화면으로 이어집니다.
- 고정저축은 출금통장에서는 나갈 금액, 입금통장에서는 들어올 금액으로 예상잔액에 반영합니다.
- 이미 잔액에 반영된 월별 입출금·이체·카드값 내역도 별도 목록으로 확인할 수 있습니다.
- 통장·소득·지출 탭의 통장별 잔액도 같은 예상잔액 화면으로 연결됩니다.

## 이전 기능 유지

- 월 소비 결제수단별 1차 목록과 예산 구분
- 월 소비달력 날짜 클릭 및 수정화면 연결
- 카드와 통장 소비 월별 통합
- 카드별 결제예정 상세
- 네모 박스 접기·펴기 및 상태 저장
- 이전 달·다음 달·이번 달 빠른 이동
- 내 예산과 목적자금 남은 금액 분리
- 일반대출·마이너스대출 이자 고정지출 자동 반영

GitHub 저장소 최상위 파일을 이 압축파일의 파일들로 덮어쓴 뒤 배포하면 됩니다.
