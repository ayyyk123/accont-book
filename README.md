# 가계부 27.3차

## 이번 수정

- 홈 상단 `월 남은 예산`에서 **내 예산 남음**과 **목적자금 남음**을 따로 표시합니다.
- 목적자금은 단순 사용금액만 보여주지 않고, 현재 실제로 사용할 수 있는 남은 금액 합계를 표시합니다.
- `월 남은 예산`을 누르면 내 예산과 목적자금의 기준·사용·남은 금액을 분리해서 확인할 수 있습니다.
- 목적자금별 남은 금액도 상세 화면에서 각각 확인할 수 있습니다.
- 예산 탭의 요약 항목도 `목적자금 사용` 대신 `목적자금 남음`으로 표시합니다.

## 27.2차 기능 유지

- 카드 결제예정액을 카드별로 묶어 표시하고, 카드를 누르면 해당 카드의 세부 사용내역을 확인할 수 있습니다.
- 일반대출과 마이너스대출 이자를 지정한 시작월부터 고정지출로 반영합니다.
- 고정지출 목록의 대출이자 이름은 `대출이자_대출이름` 형식으로 표시합니다.

GitHub 저장소 최상위 파일을 이 압축파일의 파일들로 덮어쓴 뒤 배포하면 됩니다.
