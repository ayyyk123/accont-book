# 가계부 27차

## 핵심 수정
- 월 예산 미설정 시 선택 월 소득을 자동 예산 기준으로 사용
- 남은 예산을 `예산 기준 - 내 생활비 사용 - 고정저축 예정`으로 상단과 예산 탭에서 동일 계산
- 고정수입·고정지출·고정저축, 이번 달 확인 목록, 다음 달 예정 목록을 모두 2줄 압축형으로 통일
- 상단 월 요약(남은 예산·소득·소비·저축·카드 결제예정)을 누르면 실제 상세내역 팝업 표시
- 고정수입·지출·저축 화면에 선택 월 요약을 추가하고 각 항목 클릭 시 실제 상세내역 표시
- 마이너스 통장 자동 대출 합산, 일반대출 및 대출이자 고정지출 연동 유지·보강
- 주요 금액 입력칸 쉼표 자동 표시
- 전체 백업에 현재 state 전체 저장, 검증코드·항목 개수 포함, 복원 전 자동 안전백업 및 개수 검증
- 목적자금에 선택형 저축 목표·시작금액을 추가하고 연결된 고정저축 완료액으로 진행률 계산
- 전체 검색에 고정저축·일반대출 포함
- 데이터 진단에 Firebase 연결, 중복 ID, 참조 오류, 금액 오류, 로컬 백업·복원 시험 추가
- 화면·백업·매니페스트·서비스워커 버전을 27차로 통일
