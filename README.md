# 가계부 35.7차 — All 필터 화살표 클릭 수정

## 수정한 오류

35.5차에서 표 제목 옆 필터 화살표의 `onclick` 속성 안에
한글 라벨의 큰따옴표가 그대로 들어가면서 HTML 속성이 중간에 끊겼습니다.
그 결과 화살표는 표시되지만 눌러도 필터 선택창이 열리지 않았습니다.

## 35.7차 변경

- 인라인 `onclick` 제거
- `data-all-filter-key`, `data-all-filter-label` 속성으로 변경
- 클릭 이벤트 위임 방식으로 필터창 열기
- iPhone Safari 터치 환경을 위해 `touchend` 처리 추가
- 날짜·구분·결제수단·항목·분류·금액 필터 모두 동일하게 작동
- 목적자금명·수령상태 필터도 동일하게 작동
- 적용한 필터 해제와 전체 해제 유지
- 여러 열의 필터 조합 기능 유지

## 함께 수정한 오류

- 투자 평가기준일 정규화 함수가 자기 자신을 다시 호출할 수 있던 재귀 오류 수정
- 기존 투자 데이터와 평가기준일 저장 구조 유지

## 유지 기능

- 투자 독립 탭
- 통장목록 빠른 입력
- 통장 입출금·현금계좌·투자거래 연결
- 목적자금 기간조회와 카드값 기간조회
- Firebase 데이터와 백업 구조
