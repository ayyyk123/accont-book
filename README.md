# 가계부 35.6차 — 투자 독립 탭

## 탭 구조

- `대출,저축` 바로 오른쪽에 `투자` 독립 탭 추가
- 탭 순서:
  - All
  - 고정내역
  - 카드값
  - 대출,저축
  - 투자
  - 금전거래
  - 예산
  - 통장,카드관리
  - 분류,통계
  - 검색,자산비서
  - 백업,설정
- PC 상단 탭과 모바일 하단 메뉴에 동일하게 반영

## 대출·저축 탭

- 투자 등록 버튼 제거
- 투자원금·평가금액·회수금·손익 요약 제거
- 투자 목록 제거
- 대출과 고정저축 내용만 표시
- 제목과 설명을 `대출·저축 현황`에 맞게 정리

## 투자 탭

- 투자원금
- 현재 평가금액
- 회수금
- 투자 손익
- 경매·주식·기타 투자 필터
- 진행 중·회수 완료 상태 필터
- 투자 등록·수정·삭제
- 현재 평가 기준일
- 통장 입출금과 투자거래 연결

기존 투자 데이터와 통장 입출금 연결 관계는 그대로 유지됩니다.

## 호환성

- 35.5차 All 열 필터 유지
- 투자 평가기준일과 통장목록 빠른입력 유지
- 35.4차 통장 입출금·투자거래 연동 유지
- Firebase 데이터와 백업 구조 유지
- 화면·백업·PWA 캐시를 모두 35.6차로 통일
