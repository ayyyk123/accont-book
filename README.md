# 가계부 29.5차

29.4차의 고정항목 이번 달 실제 금액 수정 기능을 유지하면서 고정수입에도 `이번 달 취소` 기능을 추가했습니다.

## 29.5차 변경사항

- 고정수입 항목에 `이번 달 취소` 버튼 추가
- 아직 입금 확인 전인 고정수입도 해당 월에만 취소 가능
- 이미 입금 확인한 고정수입을 취소하면 통장에 반영된 입금액도 함께 원상복구
- 취소한 월의 소득에서 제외
- 월 예산을 소득 기준으로 사용하는 경우 예산과 남은 금액도 자동 재계산
- `취소 해제`를 누르면 해당 월 고정수입이 다시 생성
- 다음 달 이후의 고정수입 일정은 그대로 유지
- 앱 제목, 설정, 백업 파일명, PWA 캐시 버전을 29.5차로 통일
