전체 백업

신용카드·체크카드, 통장, 분류, 카드 사용기록, 고정항목, 받을 돈·줄 돈, 월 예산과 목적자금까지 JSON 파일 하나로 저장합니다.

백업 불러오기

이전 버전의 JSON 백업과 최신 버전 백업을 불러올 수 있습니다. 기존 데이터는 최신 구조로 자동 변환됩니다.

카드 관리

신용카드는 결제일·사용기간·할부·카드값 출금을 관리합니다. 체크카드는 사용 즉시 연결 통장에서 출금되며 결제일과 할부가 적용되지 않습니다.

고정항목 관리

고정수입·고정지출을 등록하고 카드·통장·대분류·소분류·사용 상태·검색어를 조합해 필터링할 수 있습니다. 삭제 시 이번 달 자동반영 내역의 동시 삭제 여부를 선택합니다.

예산과 목적자금

내 생활비 월 예산과 데이트비·여행비 같은 목적자금을 분리합니다. 목적자금으로 지정한 카드 사용액은 카드값에는 포함되지만 내 생활비 예산에서는 제외됩니다.

통장 잔액 계산

직접 지출과 체크카드 사용은 즉시 통장 잔액에 반영됩니다. 신용카드는 카드값 납부 완료 시 통장에서 빠져 중복 차감되지 않습니다.

엑셀용 CSV

카드 사용기록, 통장내역, 분류통계를 CSV로 저장합니다. 목적자금 구분도 카드 사용 CSV에 포함됩니다.

저장 방식과 보안

로그아웃 상태에서는 현재 브라우저에 저장됩니다. 로그인 상태에서는 Firebase에 암호화된 HTTPS 연결로 동기화됩니다. 계좌 비밀번호·카드 비밀번호·주민등록번호는 메모에 적지 마세요.
